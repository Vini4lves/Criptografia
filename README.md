# Criptografia
Criptografa com Python usando cifra de César e Vigenère

Escolha entre criptografar ou descriptografar um texto.

A primeira criptografia será em Vigenère. Defina uma palavra para ser a chave.

Caso queira criptografar novamente em César, a chave gerada será aleatória.

Upper e Lower case não afetam o processo.
